*—————————————————————————————————————————————————————————使用说明———————————————————————————————*
//输入字符部分必须在提示可输入接下来操作时输入字符
1.T可开启测试模式，显示后台数据(控制输出的数组及地雷位置)，S可关闭测试模式
2.输入B可退出，R键重来
3.N为经典扫雷的左键， P为一次右键(即不带问号的小旗子)，G为两次右键(即带问号的小旗子)，F可提交当前答案，M可消除已标记点位
4.地雷个数为格子总数的五分之一
5.注意！！！！！答案必须由P标点，并且必须是以F提交形式提交，否则不予通过！！
